Upload Guidelines
  • Use UTF-8 encoded CSV files. Download the template for the correct column order.
  • Date fields accept YYYY-MM-DD, DD-MM-YYYY, or DD/MM/YYYY formats (e.g., 2024-01-15 or 15-01-2024).
  • Duplicate barcodes / lot numbers / part codes will be skipped with an error note.
  • For Devices, the lot_number must already exist in the system.
  • Rows with errors are skipped; valid rows are still inserted.
  • Lots: All GRN, invoice, GST (sgst/cgst/igst), PO, and logistics fields are optional. Numbers can include commas (e.g., 50,000).
  • For Leads: customer_type values: end_customer, corporate, individual, dealer, reseller. call_outcome values: interested, not_interested, callback, order_placed, no_answer, do_not_call.
  • Max recommended file size: 5,000 rows per upload.
